Algoritmos e Estrutura de Dados


Antes de ingressar na faculdade ja havia estudado e, na minha percepção, sabia bastante coisas sobre algumas linguagens, entre elas, a linguagem c. Após o inicio das aulas, cheguei a conclusão que sabia apenas 1% do que esta linguagem teria a oferecer. O professor João Benedito, pôde mostrar em suas aulas vários exemplos e utilizações de cada comando em certas situações. Em seguida mostrarei alguns exemplos de códigs desenvolvidos em aula.


Projetos


Tipos de dados

Este projeto foi desenvolvido para testar tipos de dados presentes na linguagem C, pedindo para que o usuário digitasse cada tipo e após isso mostrar na tela, verificando a existência de lixo de memória.
Código desenvovido no Code Blocks durante a aula


Equações do segundo grau

Este projeto tem o objetivo de realizar equações do segundo grau, em que, o usuário digita os coeficientes de a,b e c e logo em seguida o algoritmo realiza a conta.
Código desenvolvido no Code Blocks durante a aula.


Soma total

Neste projeto o intuito foi fazer uma soma de 0 até o número que o usuário digitasse, fazendo acumular toda a soma em uma variável e retornar o resultado na tela.
Código desenvolvido no Code Blocks durante a aula.


Fatorial

Este projeto, semelhante ao de cima, faz uma conta matemática. Em que, retorna o valor do fatorial do número digitado.
Código desenvolvido no Code Blocks durante a aula.


Strings

O projeto, entrando na parte das strings, mede seu tamanho, informa o primeiro e o último caracter, além de contar os espaços em branco e o total de palavras. Um resumo de várias funcionalidades em apenas um algoritmo.
Código desenvolvido no Code Blocks durante a aula.


Strings

Assim como no anterior, este algoritmo vem com varias funcionalidades envolvendo strings, porém, neste, contém também um menu de opções que da ao usuário mais possibilidades de uso, podendo ir direto ao ponto que deseja apenas selecionando qual função deseja fazer.
Código desenvolvido no Code Blocks durante a aula.


Palavras de ódio

O objetivo deste projeto foi fazer um algortimo que pudesse identificar palavras de ódio em uma frase.
Código desenvolvido no Code Blocks durante a aula.


Contador de votos

Por meio de sorteio aleatório de números, podendo ser até no máximo 100 votos, este algoritmo faz com que avalie cada cantidato e seu número de votos, exibindo assim o vencedor.
Código desenvolvido no Code Blocks durante a aula.


Registro de temperaturas

Chegando em matrizes, o seguinte algoritmo registra temperaturas durante 5 dias úteis da semana a cada 8 horas. Ao final, mostra qual a maior e a menor temperatura registrada naquela semana e seu dia específico.
Código desenvolvido no Code Blocks durante a aula.


Eleição

Por meio de vetores e matriz, este algoritmo foi desenvolvido com o intuito de eleger o candidato ganhador. Por meio de números aleatórios os votos foram gerados, posteriormente, somado todos eles, verifica-se em um vetor qual o nome do candidato com o maior total. E, por fim, mostra na tela o ganhador.
Código desenvolvido no Code Blocks durante a aula.


Registro de Alunos

O intuito deste projeto foi fazer um registro de alunos com suas repectivas notas em cada matéria, podendo ao final fazer a pesquisa do aluno que desejar.
Código desenvolvido no Code Blocks durante a aula.


Produção de grãos

O intuito deste projeto foi fazer uma estrutura de registro para armazenar a quantidade de grãos que cada produtor conseguiu na safra. Após isso, fez o calculo da menor quantidade produzida.
Código desenvolvido no Code Blocks durante a aula.


Manipulação de ponteiros

O intuito deste projeto foi dar inicio ao assunto de ponteiros, mostrando seu funcionamento na memória por meio deste programa, que, ao selecionar o endereço apontado, retorna o valor passado no código inicialmente.
Código desenvolvido no Code Blocks durante a aula.


Numeros Aleatórios

O intuito deste projeto foi dar inicio ao assunto de funcões passando parâmetro por referencia. Nele gera um número aleatório por meio de uma função.
Código desenvolvido no Code Blocks durante a aula.
