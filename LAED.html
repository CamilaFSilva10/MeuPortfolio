Laboratório de Algoritmos e Estrutura de Dados


Assim como em AED, a disciplina de Laboratório de Algoritmos e Estrutura de Dados, fez com que a linguagem C se mostrasse um mundo de novas possibilidades. Aplicando na prática exemplos de problemas e desenvolvendo a lógica computacional para resolvê-los. Alguns dos códigos desenvolvidos em aulas.


Projetos


Calendário

Este projeto foi desenvolvido com o intuito de falar o dia de amanhã. Após o usuário digitar a data desejada na tela, o algoritmo lhe informa a data do dia seguinte.
Código desenvolvido no Code Blocks durante a aula.


Contagem de caracter

Este projeto tem o objetivo de contar quantas vezes aparece cada caracter do nome digitado na tela.
Código desenvolvido no Code Blocks durante a aula.


Jogo da velha

Este projeto tem o objetivo de desenvolver um jogo da velha, podendo ser jogado com dois usuários, além disso, verticais, horizontais e diagonais, podem ser contadas como jogo ganho e se nenhum tiver todo preenchido com X ou O resulta em Velha!!
Código desenvolvido no Code Blocks durante a aula.

Registro de clientes

Este projeto foi desenvolvido para armazenar cadastros de até 100 usuários de uma loja, por meio do menu, pode ser acessado, além do cadastro, o relatório de todas as pessoas que cadastraram.
Código desenvolvido no Code Blocks durante a aula.
